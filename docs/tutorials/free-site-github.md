## خطوات إنشاء موقع مجاني بالكامل
هنا يمكنك كتابة شرح تفصيلي عن كيفية بناء الموقع خطوة بخطوة. يعتمد هذا الشرح على استخدام GitHub Pages وإطار عمل MkDocs للتوثيق والتدوين.

1. إنشاء مستودع جديد على GitHub.
2. تثبيت بايثون وأداة MkDocs على جهازك.
3. إنشاء هيكل الموقع باستخدام `mkdocs new`.

## أهم الأوامر
فيما يلي قائمة بأهم الأوامر التي ستحتاجها:

```bash
# إنشاء موقع جديد
mkdocs new my-project

# تشغيل خادم المعاينة المحلي
mkdocs serve

# بناء الموقع الثابت
mkdocs build
```
